# Merry Christmas

## Project info

A festive, single-page Christmas greeting website created to share warm wishes with friends and family.  
The site displays personalized or public Christmas messages with snowfall animation and background music, creating a joyful holiday experience.


```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
